## Project Title

Financial Dashboard (Power BI)

## Project Description

This project is a Financial Analysis Dashboard developed using Microsoft Power BI.
The goal of this report is to provide a clear and interactive view of financial performance, helping users track Revenue, Expenses, Profitability, and Trends across different time periods and business segments.

## Objectives

The main objectives of this dashboard are:

* Track overall financial performance
* Identify high-performing and low-performing categories
* Understand profit trends and expense patterns
* Support business decisions using interactive insights

## Key KPIs Covered

This report focuses on the following important financial indicators:

Total Sales / Revenue

Total Expense

Total Profit

Profit Margin (%)

Sales Growth

Expense Growth

Net Performance Trend

Common measures used in this project:

Total Expense = SUM(Financial[Expense])

Total Profit = [Total Sales] - [Total Expense]
